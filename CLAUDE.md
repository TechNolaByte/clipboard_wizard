# Clipboard Wizard

**If you are editing this project's code, read `claude-instructions-for-editing-project.md` first** —
it holds the architecture, conventions, and command roadmap.

This pointer is intentionally tiny. The app runs the `claude` CLI from `working/`, which inherits this
`CLAUDE.md`, so the full editing guide is kept out of clipboard-command context and is loaded only when
you're actually working on the project. (Do not inline the guide here with `@import` — that would pull
it back into every clipboard-command run.)
